This view provides an overview of all log files for this Joomla! instance. Clicking on one of the file names will take
you to the log file.

This view also allows you to copy, rename, download, clean up or delete the corresponding log file.

![Edit File](./_assets/logs_screenshot.png)

:::note
When a log file is deleted, it is automatically recreated by Joomla! when a new log entry is created.
:::
